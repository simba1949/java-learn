# JavaIO流

## 核心类

> 在整个 java.io 包中最重要的就是 5 个类和 3 个接口

| 类/接口      | 说明       |
| ------------ | ---------- |
| File         | 文件类     |
| InputStream  | 字节输入流 |
| OutputStream | 字节输出流 |
| Reader       | 字符输入流 |
| Writer       | 字符输出流 |
| Closeable    | 关闭流接口 |
| Flushable    | 刷新流接口 |
| Serializable | 序列号接口 |
